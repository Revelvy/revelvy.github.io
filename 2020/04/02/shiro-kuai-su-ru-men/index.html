Shiro 快速入门

Java SpringBoot Shiro
Java
发布日期:   2020-04-02
文章字数:   2.6k
阅读时长:   12 分

引言

Shiro 是一款经典的 Java 安全框架，可以在应用在 JavaSE 和 JavaEE 的场景中， 主要用于：认证、授权、加密、会话管理、Web集成以及缓存。

功能

  • Authentication：身份认证 / 登录，验证用户是不是拥有相应的身份。
  • Authorization：授权，验证某个已认证用户对某个资源是否拥有某个权限，如：验证某个用户是否拥有某个角色。
  • Session Management：会话管理，即用户登录后就是一次会话，在没有退出之前，它的所有信息都在会话中。
  • Cryptography：加密，保护数据的安全性，如密码加密存储到数据库。
  • Web Support：Web 支持。
  • Caching：缓存，比如用户登录后，其用户信息、拥有的角色 / 权限不必每次去查，提高效率。
  • Concurrency：Shiro 支持多线程应用的并发验证，如在一个线程中开启另一个线程，能把权限自动传播过去。
  • Testing：提供测试支持。
  • Run As：允许一个用户假装为另一个用户（如果他们允许）的身份进行访问。
  • Remember Me：记住我。

架构

外部

  • Subject：当前 “用户” 主体，应用代码直接交互的对象（Shiro 的对外 API 核心就是 Subject 主体），这个用户不一定是一个具体的人，与当前应用交互的任何东西都是 Subject，如网络爬虫，机器人等，所有 Subject 都绑定到 SecurityManager，与 Subject 的所有交互都会委托给 SecurityManager；可以把 Subject 认为是一个门面，SecurityManager 才是实际的执行者

  • SecurityManager：安全管理器（核心），即所有与安全有关的操作都会与 SecurityManager 交互，管理着所有 Subject，负责与其他组件进行交互，类似 SpringMVC 的 DispatcherServlet

  • Realm：领域，Shiro 从 Realm 获取安全数据（如用户、角色、权限），SecurityManager 要验证用户身份，就要从 Realm 获取相应的用户进行比较以确定用户身份是否合法；也需要从 Realm 得到用户相应的角色 / 权限进行验证用户是否能进行操作；可以把 Realm 看成 DataSource，即安全数据源。

测试

创建快速启动文件 quickStart.java 进行测试：

quickStart.java

import org.apache.shiro.SecurityUtils;
import org.apache.shiro.authc.*;
import org.apache.shiro.config.IniSecurityManagerFactory;
import org.apache.shiro.mgt.SecurityManager;
import org.apache.shiro.session.Session;
import org.apache.shiro.subject.Subject;
import org.apache.shiro.util.Factory;
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;

public class quickStart {
    private static final transient Logger log = LoggerFactory.getLogger(quickStart.class);
    public static void main(String[] args) {
        // Set up shiro environment
        Factory<SecurityManager> factory = new IniSecurityManagerFactory("classpath:shiro.ini");
        SecurityManager securityManager = factory.getInstance();
        SecurityUtils.setSecurityManager(securityManager);

        // 获取当前用户 subject
        Subject currentUser = SecurityUtils.getSubject();

        // 测试当前用户的 shiro session
        Session session = currentUser.getSession();
        session.setAttribute("someKey", "aValue");
        String value = (String) session.getAttribute("someKey");
        if (value.equals("aValue")) {
            log.info("Retrieved the correct value! [" + value + "]");
        }

        // 判断当前用户是否被认证
        if (!currentUser.isAuthenticated()) {
            // 生成随机令牌 token
            UsernamePasswordToken token = new UsernamePasswordToken("lonestarr", "vespa");
            token.setRememberMe(true);  // 记住我
            try {
                currentUser.login(token);  // 执行登陆
            } catch (UnknownAccountException uae) {
                log.info("There is no user with username of " + token.getPrincipal());
            } catch (IncorrectCredentialsException ice) {
                log.info("Password for account " + token.getPrincipal() + " was incorrect!");
            } catch (LockedAccountException lae) {
                log.info("The account for username " + token.getPrincipal() + " is locked.  " +
                        "Please contact your administrator to unlock it.");
            }
            // ... catch more exceptions here (maybe custom ones specific to your application?
            catch (AuthenticationException ae) {
                //unexpected condition?  error?
            }
        }

        //say who they are:
        //print their identifying principal (in this case, a username):
        log.info("User [" + currentUser.getPrincipal() + "] logged in successfully.");

        // 判断当前用户是否拥有角色
        if (currentUser.hasRole("schwartz")) {
            log.info("May the Schwartz be with you!");
        } else {
            log.info("Hello, mere mortal.");
        }

        // 判断当前用户是否拥有权限（粗粒度）
        if (currentUser.isPermitted("lightsaber:wield")) {
            log.info("You may use a lightsaber ring.  Use it wisely.");
        } else {
            log.info("Sorry, lightsaber rings are for schwartz masters only.");
        }

        // 判断当前用户是否拥有权限（细粒度）
        if (currentUser.isPermitted("winnebago:drive:eagle5")) {
            log.info("You are permitted to 'drive' the winnebago with license plate (id) 'eagle5'.  " +
                    "Here are the keys - have fun!");
        } else {
            log.info("Sorry, you aren't allowed to drive the 'eagle5' winnebago!");
        }
        currentUser.logout();  // 注销
        System.exit(0);
    }
}

测试

测试认证功能

1⃣ 创建域对象

在 confg 包下自定义域对象 Realm 类，配置认证过滤程序。

UserRealm.java

public class UserRealm extends AuthorizingRealm {
    // 授权
    @Override
    protected AuthorizationInfo doGetAuthorizationInfo(PrincipalCollection principals) {
        System.out.println("执行了授权~");
        return null;
    }
    // 认证
    @Override
    protected AuthenticationInfo doGetAuthenticationInfo(AuthenticationToken token) throws AuthenticationException {
        System.out.println("执行了认证~");
        // 自定义用户名 密码 （实际上从数据库中获取）
        String username = "revelvy";
        String password = "988988";
        UsernamePasswordToken usernamePasswordToken = (UsernamePasswordToken) token;
        // 用户名认证
        if (!usernamePasswordToken.getUsername().equals(username)) {
            return null;  // 抛异常：UnknownAccountException
        }
        // 密码认证（Shrio自动验证）
        return new SimpleAuthenticationInfo("",password,"");
    }
}

2⃣ 创建 Shiro 配置文件

注入域对象，配置安全管理器和过滤器工厂。在过滤器工厂中，配置 Shiro 内置认证过滤器，设置对应的过滤条件和跳转条件过滤请求，其中认证过滤权限的设置如下表：

认证权限 作用
anon 无需认证即可访问
authc 必须认证才可访问
user 拥有“记住我”才可访问
perms 拥有对某个资源的权限才可访问
role 拥有某个角色权限才可访问

ShiroConfig.java

import org.apache.shiro.spring.web.ShiroFilterFactoryBean;
import org.apache.shiro.web.mgt.DefaultWebSecurityManager;
import org.apache.shiro.mgt.SecurityManager;
import org.springframework.beans.factory.annotation.Qualifier;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

import java.util.LinkedHashMap;
import java.util.Map;

@Configuration
public class ShiroConfig {
    // 1. Realm 对象
    @Bean
    public UserRealm userRealm() {
        return new UserRealm();
    }
    // 2. securityManager 安全管理器
    @Bean(name = "securityManager")
    public SecurityManager securityManager(@Qualifier("userRealm") UserRealm userRealm) {
        DefaultWebSecurityManager securityManager = new DefaultWebSecurityManager();
        securityManager.setRealm(userRealm);  // 关联 Realm 对象
        return securityManager;
    }
    // 3. ShiroFilterFactoryBean Shiro过滤器工厂
    @Bean
    public ShiroFilterFactoryBean getShiroFilterFactoryBean(@Qualifier("securityManager") SecurityManager securityManager) {
        ShiroFilterFactoryBean bean = new ShiroFilterFactoryBean();
        bean.setSecurityManager(securityManager);  // 关联 securityManager 对象
        Map<String, String> map = new LinkedHashMap<>();
        map.put("/user/*", "authc");
        bean.setFilterChainDefinitionMap(map);
        bean.setLoginUrl("/toLogin");  // 登陆页面（未认证跳转页面）
        return bean;
    }
}

3⃣ 创建接口测试

在创建的登陆测试页面中，输入用户名和密码执行登陆，Shiro 会帮我们自动校验，可在控制台查看登录结果。

HelloController.java

@Controller
public class HelloController {
    @GetMapping("/toLogin")
    public String toLogin() {
        return "login";
    }
    @GetMapping("/doLogin")
    public String doLogin(String username, String password) {
        // 获取当前用户 subject
        Subject subject = SecurityUtils.getSubject();
        // 生成随机令牌 token
        UsernamePasswordToken token = new UsernamePasswordToken(username, password);
        try {
            subject.login(token);  // 执行登陆
            System.out.println("登陆成功~");
            return "hello";
        } catch (UnknownAccountException uae) {
            System.out.println("用户名错误~");
            return "login";
        } catch (IncorrectCredentialsException ice) {
            System.out.println("密码错误~");
            return "login";
        }
    }
}

测试认证功能（数据库）

实际开发中，当我们都需要连接数据库时，需配置域对象：

UserRealm.java

public class UserRealm extends AuthorizingRealm {
    @Autowired
    UserService userService;
    // 认证
    @Override
    protected AuthenticationInfo doGetAuthenticationInfo(AuthenticationToken token) throws AuthenticationException {
        System.out.println("执行了认证~");
        UsernamePasswordToken usernamePasswordToken = (UsernamePasswordToken) token;
        // 从数据库读取用户名 密码
        User user = userService.queryUserByName(usernamePasswordToken.getUsername());
        // 用户名验证
        if (user == null) {
            return null;  // 抛异常：UnknownAccountException
        }
        // 密码验证（Shrio自动验证）
        // 可以加密：MD5：dasdsd11 MD5盐值加密：dasdsd11username
        return new SimpleAuthenticationInfo("", user.getPwd(), "");
    }
}

测试授权功能（数据库）

测试完认证功能，我们接下来就对授权功能一探究竟。

1⃣ 域对象

UserRealm.java

public class UserRealm extends AuthorizingRealm {
    @Autowired
    UserService userService;
    // 认证
    @Override
    protected AuthenticationInfo doGetAuthenticationInfo(AuthenticationToken token) throws AuthenticationException {
        System.out.println("执行了认证~");
        UsernamePasswordToken usernamePasswordToken = (UsernamePasswordToken) token;
        // 从数据库读取用户名 密码
        User user = userService.queryUserByName(usernamePasswordToken.getUsername());
        // 用户名验证
        if (user == null) {
            return null;  // 抛异常：UnknownAccountException
        }
        // 密码验证（Shrio自动验证）
        // 可以加密：MD5：dasdsd11 MD5盐值加密：dasdsd11username
//        return new SimpleAuthenticationInfo("", user.getPwd(), "");
        // 传递当前用户方便授权取用
        return new SimpleAuthenticationInfo(user, user.getPwd(), "");
    }
    // 授权
    @Override
    protected AuthorizationInfo doGetAuthorizationInfo(PrincipalCollection principals) {
        System.out.println("执行了授权~");
        SimpleAuthorizationInfo info = new SimpleAuthorizationInfo();
        // 获取当前用户 subject
        Subject subject = SecurityUtils.getSubject();
        User user = (User) subject.getPrincipal();
        // 设置权限（从数据库中读取该用户的权限）
        info.addStringPermission(user.getPerms());
        return info;
    }
}

2⃣ Shiro 配置文件

ShiroConfig.java

@Configuration
public class ShiroConfig {
    // 3. ShiroFilterFactoryBean Shiro过滤器工厂，设置对应的过滤条件和跳转条件过滤请求
    @Bean
    public ShiroFilterFactoryBean getShiroFilterFactoryBean(@Qualifier("securityManager") SecurityManager securityManager) {
        ShiroFilterFactoryBean bean = new ShiroFilterFactoryBean();
        bean.setSecurityManager(securityManager);
        Map<String, String> map = new LinkedHashMap<>();
        map.put("/user/add","perms[user:add]");  // 授权必须在认证之前
        map.put("/user/*", "authc");
        bean.setFilterChainDefinitionMap(map);
        bean.setUnauthorizedUrl("/noAuth");  // 未授权页面
        bean.setLoginUrl("/toLogin");
        return bean;
    }
}
